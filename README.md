# Euclidean_Algorithm_GCD_LCM
Euclidean algorithm is useful for computing GCD (Greatest Common Divisor) and LCM (Least Common Multiple) to run the programs more faster and efficient.

# Input and output examples:
Input 1:

4 6

Output 1:

GCD = 2

LCM = 12

Input 2:

17 8

Output 2:

GCD = 1

LCM = 136
